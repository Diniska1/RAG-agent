# RAG-agent
В данном репозитории содержится реализация RAG агента, реализованного на LangGraph , который ищет информацию из ограниченного списка статей википедии и возвращает ответ.

Также с помощью специального инструмента может быть вычислено арифметическое выражение.

Схема графа представлена на картинке RAG_graph.png.

Во второй ячейке файла .ipynb (либо в секретах) необходимо вставить ваш API_KEY.
